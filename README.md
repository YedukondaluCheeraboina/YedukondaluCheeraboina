An end-to-end Crime Analytics Platform built using **Python, Streamlit, MySQL, SQLAlchemy, Pandas, Plotly, and ReportLab**. The project demonstrates the complete analytics lifecycle, including ETL processing, interactive dashboards, business intelligence reporting, and automated report generation.

* ✅ Interactive Streamlit Dashboard
